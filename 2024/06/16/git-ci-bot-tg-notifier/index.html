仓库 Secret 设置

前文提到了报错。这是正常的，因为你没有为这个仓库设置必要的secret。

设置 Bot Token

  1. 从 BotFather 那里获取你创建的 Bot 的Token

  • 网上都是答案所以我就不写了哟～

  1. 打开仓库设置里的 Secret 页面，选择添加 Secret

  • Secret 名字为 TELEGRAM_TO。

  • 内容为你的 Bot 的 Token。
